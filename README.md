# PyGRPY

Python port of Generalized Rotne Prager Yakamawa hydrodynamic tensors.

Now also supports jax, and jax.grad.


# Original code

Original code in Fortran90 by Pawel Jan Zuk here:
https://github.com/pjzuk/GRPY
